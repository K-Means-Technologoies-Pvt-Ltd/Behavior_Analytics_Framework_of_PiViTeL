# 4. Missing Value Analysis

Missing values were inspected for all dataset attributes.

### Key Observations

1. Object detection related attributes contain missing values in **1098 rows**, including:

- detection_id
- tracking_id
- class
- confidence
- bounding box coordinates
- object position
- object distance

These missing values likely occur when no object was detected in a frame.

---

2. Several overlay-related attributes contain **100% missing values (4398 rows)**:

- overlay_time
- overlay_date
- overlay_gps
- overlay_accel_g
- overlay_gyro_deg_s

These fields may represent optional overlay metadata that was not recorded during telemetry collection.

---

3. GPS attributes contain minor missing values:
