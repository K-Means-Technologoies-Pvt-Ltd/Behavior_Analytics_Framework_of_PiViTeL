# Behavior_Analytics_Framework_of_PiViTeL

## Project Title
An Explainable Driving Behavior Analytics Framework for Government-Aligned Pay-As-You-Drive Models Using PiViTel Edge Telemetry

## Description

This project aims to design a data analytics and visualization framework for analyzing vehicular telemetry generated by the PiViTel edge device.

The PiViTel device captures real-time driving data including:

- frame number
- timestamp
- vehicle speed
- object detection confidence
- object class
- object distance
- GPS latitude and longitude
- spatial object position

The project transforms raw telemetry data into meaningful driving behavior indicators aligned with Pay-As-You-Drive (PAYD) models promoted in modern telematics-based insurance systems.

## Objectives

- Analyze PiViTel-generated vehicular telemetry data
- Extract behavioral indicators such as overspeed events and proximity risks
- Compute driving risk metrics
- Develop an explainable driving score model
- Visualize driving patterns using statistical and spatial visualization techniques
